# 暗黑4死灵法师装备升级指引

这是一个可直接打开的中文交互网页，面向暗黑4赛季服、有资料片、已能刷折磨的死灵法师账号。

## 打开方式

直接双击 `index.html` 即可使用。页面不需要安装依赖，也不需要联网运行。

## 更新资料

- 流派、装备、词条、刷装地点都在 `data.js`。
- 背景图在 `assets/necromancer-banner.png`。
- 赛季或 Boss 掉落更新时，优先改 `data.js`，页面会自动按新数据渲染。

## 资料口径

内容按 2026-06-09 的暗黑4第13赛季资料口径整理，并附有官方、Wowhead 与 Mobalytics 来源链接。赛季中期补丁可能改变强度和掉落表，使用前建议核对页面底部来源。
